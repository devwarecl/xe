
## What is XE?
XE is a framework, written in C++17 for building high-performance and cross-platform multimedia applications.

## Preliminary Supported Platforms
* Windows 7+
* Linux (X Window System and Wayland)
* HTML5 
* Android

## Supported Compilers
* Visual C++
* GCC
* clang
